# 协作流

虾饺提供三层协作能力：**协作链** → **可视化面板** → **工作流引擎**。

</p>

## 协作链

在群组中配置一条 Agent 接力序列：

```
小说家 → 编辑 → 翻译官
```

用户发一句"写一首关于春天的诗"，整条链自动接力：

1. **小说家** 创作诗歌
2. **编辑** 自动接收诗歌并润色
3. **翻译官** 自动接收润色后的诗歌并翻译

每个 Agent 的输出成为下一个 Agent 的输入。

## 可视化面板

协作链运行时，界面上显示一个实时状态面板：

- 每个 Agent 节点的状态（等待中 / 运行中 / 已完成 / 出错）
- 各环节的输出内容
- 时间线和进度条

支持的交互：

| 操作 | 说明 |
|------|------|
| **人工确认** | 在自动接力前暂停，等待你确认后再继续 |
| **终止** | 随时停止整条链 |
| **编辑** | 修改中间结果后再传给下一个 Agent |
| **历史回放** | 查看过去的协作链运行记录 |

## 工作流引擎

工作流是比协作链更强大的编排机制：

- **条件分支** — 根据中间结果走不同路径
- **错误处理** — 重试 / 跳过 / 回退
- **人工审批** — 关键节点等待人工确认
- **多步骤** — 不限于线性，支持复杂流程

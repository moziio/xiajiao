### 2.2 创建仓库 + 首次推送

```powershell
# 1. 进入项目目录
Set-Location "C:\Users\admin\.openclaw\web-im-client"

# 2. 初始化 Git
git init

# 3. 暂存所有文件（.gitignore 自动排除敏感文件）
git add .

# 4. 检查！确认没有敏感文件
git status
# ⚠️ 确认以下文件 NOT 在列表中：
#   models.json, agents.json, im-settings.json, data/*, *.bak

# 5. 首次提交
git commit -m "feat: xiajiao-im v5.0.0 — AI Agent team collaboration platform"

# 6. 在 GitHub 创建仓库（用 gh CLI）
gh repo create xiajiao-im --public --description "AI Agent 团队协作平台 — 4 个依赖，npm start，管理你的 AI 团队" --source . --push

# 或者手动在 github.com 上创建，然后：
git remote add origin https://github.com/<你的用户名>/xiajiao-im.git
git branch -M main
git push -u origin main
```

## 风险与应对

| 风险 | 概率 | 应对 |

## 最关键的一件事

> **不要等到完美再发布。现在就是最好的时机。**
>
> v5.0.0 已经是一个完整可用的产品。每多等一天，被人抢先的风险就多一分。
> 开源项目的成功 = 产品力 × 推广力 × 时间。你的产品力已经够了，现在需要的是推广力和时间。
